# CrimeVista

**CrimeVista** is an AI-Driven Crime Analytics & Visualization Platform designed to help law enforcement agencies transform fragmented crime records into actionable intelligence. The platform enables investigators and analysts to move beyond traditional spreadsheet-based reporting by providing interactive dashboards, geospatial crime visualization, criminological network analysis, behavioral pattern discovery, and AI-powered predictive insights.

Built for the **Karnataka State Police (KSP)** and the **State Crime Records Bureau (SCRB)** hackathon challenge, CrimeVista integrates crime incidents, suspects, victims, locations, and socio-economic indicators into a unified intelligence platform. It empowers investigators to identify crime hotspots, uncover hidden criminal associations, analyze repeat offender behavior, detect anomalies, and forecast emerging crime risks through data-driven analytics.

The platform focuses on transforming raw crime data into meaningful visual intelligence, enabling proactive policing, strategic decision-making, and evidence-based resource allocation while remaining aligned with the requirements of the hackathon problem statement.

## Key Features

* Interactive crime analytics dashboards
* District and police station-level drill-down analysis
* Geospatial crime hotspot visualization
* Spatiotemporal crime pattern analysis
* Criminal network and relationship mapping
* Repeat offender and Modus Operandi (MO) analysis
* Hidden association detection
* Socio-economic correlation dashboards
* AI-powered predictive risk scoring
* Crime anomaly detection
* Statistical pattern and trend discovery

**Tech Stack:** React • Node.js • Python • PostgreSQL • GIS Mapping • Machine Learning
